# 🛒 Assignment 5 – FastAPI Cart & Checkout System

## 📌 Overview
This project is part of Assignment 5, where a simple **Cart and Checkout API** is built using **FastAPI**.  
It allows users to add items to a cart, view the cart, and perform checkout with total calculation.

---

## 🚀 Features
- Add products to cart
- View cart items
- Calculate total price
- Checkout functionality
- Generates order summary

---

## 🛠️ Tech Stack
- Python 3.x
- FastAPI
- Uvicorn
- Pydantic
